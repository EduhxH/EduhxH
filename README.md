## About

18-year-old developer building across the full stack — from REST APIs with proper auth and validation layers to AI agent systems with tool orchestration, MCP servers, and RAG pipelines.

On the AI side, I build agents that reason through tool use, MCP servers that bridge LLMs to external services (Pinecone, Gmail, Google Cloud, IDEs), and local RAG pipelines with semantic chunking and anti-hallucination retrieval. On the web side, I ship full-stack applications with typed frontends (Next.js + TypeScript), Python/Node backends, and production deployments.

I care about the decisions behind the code — why a relevance threshold before generation, why Zod at the API boundary, why FastMCP over raw MCP. Currently expanding into **C++** and **Java**.
